04 / Workspace

HxA 时代的 Slack

Slack 解决了人与人的协作。
Workspace 解决的是人与 Agent、Agent 与 Agent 的协作。

产品目标

  • 可视化与可控 — 从 DOS 到 Windows，Agent 状态全程透明，看得见、管得住、量得出
  • 人机混合协作 — 人和 Agent 在同一空间协作，不是人用工具，是团队协同
  • Agent 高度专业化 — 模板市场 + Fork 最优实践 + OAuth 一键授权，2 分钟从零到能干活
  • 多 Agent 协同 — 支撑三种工作负载：简单、复杂、资源密集
  • 组织知识沉淀 — 知识归组织不归个人，Agent 离职不带走经验

05 / 设计理念

Agent Native 设计

产品从设计之初就把 Agent 当成原生参与者，不是事后集成。

层次一：Agent 辅助人使用产品
Copilot 悬浮助手 • Slash Commands • Smart Reply • 操作代理（L1-L4 分级权限）
层次二：产品对 Agent Friendly
Agent Action Protocol（按钮/表单/卡片）• Context API • Event Subscription — Agent 驱动完整交互
层次三：产品本身就是 Agent（SaaA）
软件具备 Agent 能力 — 主动响应事件、自动执行任务、跨系统协调。人类 UI 和 Agent API 是同一产品的两个界面。
